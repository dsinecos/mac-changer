## Usage

```
Usage: mac-changer.py [OPTIONS]

  CLI Tool to change MAC Address of an interface

Options:
  -i, --interface TEXT  Specify the interface for which you'd like to change
                        the MAC address
  -m, --mac TEXT        Specify the new MAC Addr to be assigned to the
                        interface
  --help                Show this message and exit.

```
